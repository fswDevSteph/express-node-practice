Express JS / Node Project Steps:

  1. Create a folder to hold your Project
  2. Open that folder in your IDE and then open the terminal
  3. Run, 'npm i -y' to start a new express project. This creates a package.json file.
  4. Run, 'npm i --save-dev nodemon' This will install nodemon as a dev dependency. This will restart the server automatically when changes are detected preventing the need to do it manually each time.
